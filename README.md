## Why I made it:

Let's be brutally honest: almost every basic notepad out there is absolute garbage. They are either incredibly ugly, horribly outdated, or bloated with complicated features you don't actually need. I got completely tired of using terrible text editors, so I decided to build my own—and make it *actually* good.

I wanted a notepad that looks gorgeous (like a modern app should), runs smoothly, and has essential features like markdown, tabs, and password protection, without sacrificing simplicity. So, I made Simplepad.

---

## Features

- **Modern UI:** Sleek design with smooth crossfade animations and customizable Dark/Light modes.
- **Tabbed Interface:** Manage up to 9 notes simultaneously with quick `Ctrl + 1-9` shortcuts.
- **Local File Support:** Import and export your notes natively as `.txt` files directly to your PC.
- **Note Locking:** Protect your secrets! Set a password for individual notes.
- **Markdown Support:** Live text formatting preview (Bold, Italics, Lists, Headers).
- **Customization:** Tag your notes, change their accent colors, and favorite the important ones.
- **Auto-Updates:** Automatically checks the GitHub repository for new versions and installs them seamlessly.
